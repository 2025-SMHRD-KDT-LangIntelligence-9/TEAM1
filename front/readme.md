ToneGuard 프로젝트 실행 가이드
본 프로젝트는 AI를 활용한 실시간 말투 교정 및 소통 보조 도구인 **'ToneGuard'**의 웹-데스크톱 통합 프로토타입입니다. 팀원 여러분은 아래 절차에 따라 환경을 설정해 주세요.

📋 1. 환경 요구사항 (Environment)
언어: Python 3.13 이상
운영체제: Windows 10/11 권장 (데스크톱 GUI 제어용)
주요 라이브러리: FastAPI, CustomTkinter, PyGetWindow

📂 2. 프로젝트 구조 (Project Structure)
Plaintext
project1/
├── main_web.py           # 웹 서버 (FastAPI 관제탑)
├── scanner_window.py     # 데스크톱 스캐너 앱 (GUI)
├── requirements.txt      # 필수 라이브러리 목록
├── templates/            # HTML 파일 모음 (index, login, mypage)
└── static/               # 이미지 및 로고 (logo.png 등)
🛠️ 3. 초기 설정 (Setup)
VS Code 터미널에서 아래 명령어를 순서대로 입력하여 필요한 도구들을 설치합니다.

1) 필수 패키지 설치
pip install fastapi uvicorn jinja2 customtkinter pygetwindow pyautogui pyperclip requests
(선택사항) 만약 requirements.txt 파일이 있다면 아래 명령어로 한 번에 설치 가능합니다:
pip install -r requirements.txt
🚀 4. 실행 방법 (How to Run)
1) 서버 가동
터미널에서 웹 서버를 실행합니다.
python main_web.py
2) 접속
브라우저를 열고 아래 주소로 접속합니다.
URL: http://192.168.0.6:8888
3) 시나리오
로그인: 이메일/비번 입력
스캔 시작: 메인 화면의 [Scan] 버튼 클릭
웹 브라우저가 자동으로 최소화되고, 모니터 우측 상단에 데스크톱 스캐너가 실행되는지 확인합니다.
메인 복귀: 스캐너 창의 [메인] 버튼 클릭
스캐너가 종료되고 숨겨져 있던 웹 브라우저가 다시 나타나는지 확인합니다.
✨ 5. 주요 기능 안내
Hybrid Workflow: 웹 서버가 시스템 프로세스를 제어하여 데스크톱 창을 띄우는 하이브리드 방식 구현.
Seamless UI: 브라우저 최소화/복구 기능을 통한 매끄러운 사용자 경험 제공.
DPI Awareness: 고해상도 모니터에서도 UI 배율이 깨지지 않도록 시스템 패치 적용.
⚠️ 주의사항
포트 충돌: 서버 실행 시 [Errno 10048] 에러가 발생하면, 터미널에 taskkill /f /im python.exe를 입력하여 기존 프로세스를 완전히 종료한 뒤 다시 실행해 주세요.

창 위치: 스캐너 창은 실행 후 약 0.2초 뒤에 우측 상단으로 이동하도록 설계되어 있습니다.
# 🛡️ 톤가드(ToneGuard) 사용자 가이드
직장 내 커뮤니케이션 코칭 에이전트, **톤가드(ToneGuard)**의 웹 서비스 및 데스크톱 스캐너 사용 방법입니다.

## 1. 시작하기 (로그인 및 설치)
### 1-1. 웹사이트 접속 및 로그인
1. 톤가드 웹사이트에 접속합니다.
2. 회원가입 시 등록한 이메일과 비밀번호를 입력하여 **로그인**합니다.
   * *Tip: '이메일 기억하기'를 체크하면 다음 접속 시 비밀번호만 입력하여 빠르게 로그인할 수 있습니다.*

### 1-2. 스캐너 실행 및 설치 (최초 1회)
1. 메인 화면 중앙에 있는 파란색 **[스캔 시작하기]** 버튼을 클릭합니다.
2. **PC에 스캐너가 설치되어 있지 않은 경우:**
   * "스캐너 연결 실패" 팝업이 나타납니다.
   * 안내에 따라 **[스캐너 설치 프로그램 다운로드]** 버튼을 눌러 `ToneGuard_Installer.exe`를 다운로드하고 실행합니다.
   * *(설치 시 필요한 윈도우 필수 패키지까지 자동으로 설치되므로 안심하고 진행하셔도 됩니다.)*
3. **설치가 완료된 후:**
   * 다시 웹사이트에서 **[스캔 시작하기]** 버튼을 누르면, 바탕화면 우측 하단에 톤가드 스캐너 창이 나타납니다.

## 2. 스캐너 활용하기 (실시간 코칭)
### 2-1. 스마트 감지 모드
* 톤가드 스캐너는 불필요한 리소스 낭비를 막기 위해 **스마트 감지 로직**이 적용되어 있습니다.
* 일반 웹 브라우저나 엑셀 등을 사용할 때는 스캐너가 대기 상태를 유지합니다.
* **카카오톡, 슬랙(Slack), 텔레그램** 등 업무용 메신저 창을 클릭하여 활성화하면 스캐너가 즉시 감지 모드로 전환되며 상태창에 `✅ 카카오톡 감지됨!`이 표시됩니다.

### 2-2. 메시지 작성 및 AI 분석
1. 메신저 창에 상대방에게 보낼 메시지를 자유롭게 작성합니다.
2. 작성을 마치고 **약 2초간 키보드 입력을 멈추면**, 톤가드가 텍스트를 자동 인식합니다.
3. 스캐너의 버튼이 임시로 **[분석 중...]**으로 변경되며 AI가 상황에 맞는 문장을 생성합니다. (최대 5초 소요)

### 2-3. 교정 문장 적용하기
1. 분석이 완료되면 마우스 커서 근처에 **[톤가드 AI 팝업창]**이 나타납니다.
2. AI가 분석한 대화의 성격(예: 업무, 의견충돌 등)과 함께 3가지 톤(정중하게, 친근하게, 단호하게)의 교정 문구가 제시됩니다.
3. 마음에 드는 **문장 버튼을 클릭**합니다.
4. 팝업창이 닫히며, **메신저 입력창에 선택한 교정 문장이 자동으로 입력(붙여넣기)** 됩니다. 바로 전송 버튼을 눌러 대화를 이어가세요!

## 3. 스캐너 수동 제어
바탕화면에 떠 있는 톤가드 스캐너 창에서 다음 기능들을 조작할 수 있습니다.
* **AUTO 스위치:** 스위치를 끄면 자동 감지 및 팝업 기능이 일시 정지됩니다.
* **스캔 시작 / STOP 버튼:** 클릭하여 스캐너의 전체 동작을 수동으로 켜거나 끌 수 있습니다.
* **투명도 슬라이더:** 우측 상단의 슬라이더를 조절하여 스캐너 창의 투명도를 변경할 수 있습니다. (업무 시 시야를 가리지 않도록 조절)
* **Main 버튼:** 스캐너 창을 숨기고 트레이(백그라운드) 상태로 전환합니다. 

## 4. 설정 및 관리 (마이페이지)
웹사이트 상단 메뉴의 **[My Page]**에서 개인 설정 및 기록을 관리할 수 있습니다.
* **나의 교정 기록:** AI가 교정해 준 과거의 원문과 교정문, 톤(말투) 기록을 확인할 수 있습니다. 불필요한 기록은 우측의 [X] 버튼을 눌러 삭제할 수 있습니다.
* **정보 수정:** 현재 비밀번호를 확인한 뒤 안전한 새 비밀번호로 변경할 수 있습니다.
* **대화 기록 저장 동의:** * AI 품질 향상을 위한 텍스트 저장 옵션을 스위치로 켜고 끌 수 있습니다.
  * *이 스위치를 조작하면 켜져 있는 데스크톱 스캐너에도 설정이 **실시간으로 동기화**됩니다.*
* **회원 탈퇴:** 더 이상 톤가드를 사용하지 않을 경우, 모든 데이터(교정 기록 포함)를 영구적으로 삭제하고 탈퇴할 수 있습니다.
